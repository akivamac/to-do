Get the App

We don't offer App Store access, but you can install Peaceful Tasks directly from here. Once installed, it works offline and launches from your home screen like a native app. On Android you may see a prompt to allow installation from unknown sources — just follow the popup that appears and you'll be all set.

Install on Android

Chrome or Firefox will show an install prompt. Just tap "Install" when you see it.

If you don't see a prompt, or it's already installed:

  1. Tap the menu (three dots) in your browser
  2. Select "Install app" or "Add to Home screen"

Install on iOS

  1. Open this page in Safari
  2. Tap the Share button (box with arrow)
  3. Scroll down and tap "Add to Home Screen"
  4. Confirm with "Add"

Peaceful Tasks will now appear on your home screen like a regular app!

Install on macOS

Use Chrome, Edge, or Brave for the best experience:

  1. Open akivamac.github.io/to-do in Chrome, Edge, or Brave
  2. Look for the install icon (a monitor with a down arrow) on the right side of the address bar
  3. Click it, then click "Install" in the popup
  4. Peaceful Tasks will appear in your Applications folder and Dock

Don't see the install icon? Try:

  1. Click the three-dot menu (⋮) in Chrome or Edge
  2. Select "Cast, save, and share" → "Install page as app"
